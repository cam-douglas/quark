# Quark Task Store

Path: `state/tasks/`

This directory houses all YAML task lists, archived Markdown snapshots, and test helpers used by the Quark State System.

## Current Contents (2025-09-01)
```
state/tasks/
├─ tasks_high.yaml            # Auto-generated high-priority roadmap tasks
├─ tasks_medium.yaml          # Auto-generated medium-priority tasks
├─ tasks_low.yaml             # Auto-generated low-priority tasks
├─ tasks_archive.yaml         # Completed tasks archived by the state system
├─ chat_tasks_*.yaml          # Ad-hoc chat tasks (auto-generated)
│
├─ archive/                   # Historical Markdown backups (read-only)
│   └── COMPLEXITY_EVOLUTION_TODOS.md.backup
│   └── STAGE_N3_EVOLUTION_IMPLEMENTATION_SUMMARY.md.backup
│
├─ integrations/README.md     # Legacy docs kept for reference
└─ tests/integration_tasks/   # Python tests exercising task integrations
```

### Workflow
1. Roadmap tasks are pulled via `QUARK_STATE_SYSTEM.generate_recommendations()` and written to the YAML files above.
2. When you create chat-tasks the system writes a new `chat_tasks_<title>_N.yaml` file here.
3. Completed tasks are marked `DONE` and moved into `tasks_archive.yaml` automatically.

### Maintenance
• Do **not** manually edit the autogenerated YAML files; use the CLI commands in `QUARK_STATE_SYSTEM.py` instead.
• Archive Markdown files are preserved for historical traceability and should not be altered.
